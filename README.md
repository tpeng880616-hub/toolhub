## 🚀 快速开始

1. 克隆仓库
```bash
git clone https://github.com/yourusername/toolhub.git
```

2. 部署到 GitHub Pages
- 进入仓库 Settings
- 找到 Pages 选项
- Source 选择 main 分支
- 保存后即可访问

## 📺 Google AdSense 配置

1. 申请 [Google AdSense](https://www.google.com/adsense/)
2. 获取广告代码
3. 替换 `index.html` 中的以下占位符：
   - `YOUR_ADSENSE_ID` → 您的 AdSense ID（如 `ca-pub-123456789`）
   - `YOUR_AD_SLOT` → 广告位 ID
4. 重新部署网站
